FONDY payment gateway integration for Paid memberships Pro.

== Description ==
This Paid memberships Pro payment plugin adds Flitt as payment gateway to plugin Paid memberships Pro. Flitt can accept VISA and MasterCard securely, quickly and easily on your store in minutes. Recurring payments, simple, straightforward pricing, top notch fraud intelligence, and 24/7 support.

= Start with Flitt =
[Create Free Flitt Account](https://portal.flitt.com/mportal/#)

== Features ==
* Customize your checkout window ([Some styles](http://jsfiddle.net/8h65h91t/))
* Easy configuration - only Merchant ID and Secret Key!

== Allowed currencies ==
With us your customers can make purchases in many currencies. Flitt support  GEL, EUR, USD, MDL, AMD, UZS, AZN, KZT.

== Tariffs ==
Only commission with payment. It is possible to adjust payment of the commission on itself, or to impose on the client.

[See current tariffs on the Flitt website](https://flitt.com)


== Installation ==
1. Ensure you have latest version of plugin installed (Paid memberships Pro)
2. Upload the plugin files to the `/wp-content/plugins/`(or upload the plugin zip file using wp-admin.)
3. Activate the plugin through the 'Plugins' menu.
